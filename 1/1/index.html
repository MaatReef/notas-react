1.1 - ¿Qué es react? 

    --------------------------------------------------------
    Librería o Bibliteca:
    
    Para crear Interfaces de Usuarios de Manera Dinámica.. 	Con Css y Html (Se crean de Manera estática)
                                Con Js, nos apoyamos en el dinamismo.
    
    Con React Por medio de Js, podemos crear Aplicaciones Dinámicas.
    
        - Declarativo: Su sintaxis, a nivel visual es sencilla de entender, al declarar señalamos que sucederá en esa interfaz de Usuario.
                   Legible, fácil de entender.
    
        - Basado en Componentes: Encapsulados que manejan su propio estado.
    
        - Aprende una vez, escríbelo donde sea: Al aprender React, podemos re utilizar el código para emplearlo en Native, para móvil.
                - React puede también renderizar desde el servidor usando Node, así como potencializar aplicaciones móviles usando React Native.
                - Next o Gaspy, construir aplicaciones desde el Servidor.

1.2 - Construye tu propia librería js 

        --------------------------------------------------------
        Construiremos una librería para entender más de cerca a React.. 
        Nuestra librerìa construirá interfaces de Usuarios.
        --------------------------------------------------------
            En Vs code..
        Creamos una carpeta donde deseemos.. Curso-React
            Nueva carpeta dentro.. LIBRERIAJS
        
        Creamos un archivo index.html	-> Html5
        
        
        
        < body>
            < header>
                < img src="./images/logo.svg" alt="Logo Libreria Js">
                < nav class="main-menu">
                    < ul>
                        < li> < a href="/">Inicio 
                        < li> < a href="/nosotros">Nosotros < /li>
                        < li> < a href="/contacto">Contacto < /li>			-> Este contenido se verá de manera Estática.
                    
                
            < header>
            < script src="./libreria.js" type="module">< /script>
        									->"./" 	-> Indica la carpeta actual
        
        --------------------------------------
             Ahora en el html, borramos todo e intentaremos generar lo mismo, desde js.. 
        --------------------------------------
        
        < body>
            < div id="root">
        
            < script src="./libreria.js" type="module">
        < /body>
        
        --------------------------------------
            En el Archivo.. libreria.js		-> Para crear lo mismo de Js, sería un tanto "tedioso"
                                -> Lo estático lo hacemos de manera dinámica con Js.
        --------------------------------------
        
        let header = document.createElement("header")
        header.id = "mainHeader"
        header.classList.add("main-header")
        
        let root = document.getElementById("root)
        root.appendChild(header)				-> Le agregamos un hijo (header)
        
        *Si quisieramos crear otros componentes deberiamos repetir esto indefinidamente, por lo cual es necesario crear nuestras propias funciones*
        
        --------------------------------------------------------
        
        function createElement(elem, attr = {}) {				-> Armamos un objeto con los atributos del elemento a crear.
        
            let element = document.createElement(elem)
            element.id = attr.id						- Los atributos del objeto se los agregamos con un -> .
            element.classList.add(attr.className)				-> Evitar usar class que puede ser de POO
            
            return element
        }
        
        function render(elem, DOMelem) {
            DOMelem.appendChild(elem)
        }
        
        const Header = createElement("header", { id: "mainHeader2", className: "main-header-2" })
        render(Header, document.getElementById("root"))

1.3 - Optimizando nuestra librería js

        --------------------------------------------------------
        De esta manera creamos funciones que luego podremos re utilizar las veces que deseemos.
        --------------------------------------------------------
        Creamos un nuevo archivo llamado script.js..

        
        import { createElement, render} from "./liberia.js"						-> Importamos las funciones a utilizar.
        
        const Header = createElement("header", { id: "mainHeader2", className: "main-header-2" })
        render(Header, document.getElementById("root"))
        
        --------------------------------------------------------
        Y desde el archivo libreria.js.. Exportamos la función.

        
        export function createElement(elem, attr = {}) {				
        
            let element = document.createElement(elem)
            element.id = attr.id						
            element.classList.add(attr.className)				
            
            return element
        }
        
        export function render(elem, DOMelem) {
            DOMelem.appendChild(elem)
        }
        
        --------------------------------------------------------
        Y desde el index.html.. Llamamos al script.

        
            < script src="./script.js" type="module">< /script>
